Technical Analyst / SQL Assessment

  1. Home
  2. SQL Assessment

This assessment simulates a real-world project that you might deliver in this role, though simplified. As part of this assessment you will:

  • Create a stored procedure to report on some data;
  • Perform a number of data import and manipulation tasks using T-SQL;
  • Prepare an outcome report listing any errors or issues you identified;

When complete, our technical team will review your solution and may schedule a short follow-up interview to discuss your approach and to answer any technical questions that you have. We'll learn about your abilities to work with data, and you'll learn about how we appproach collaboration and technical problem solving. It's a win-win!

We know your time is valuable, so we're only asking for 2 to 3 hours on this. If you can't finish the whole thing, that's OK! Do what you can. (You're welcome to spend more time if you want, but it's not required.)

Want to use your own computer?

If you have your own SQL Server environment then you can can do this assessment there if you prefer. Just clone the Heuristics.SqlEval GitHub repo.

Accessing the local database

This workspace already has SQL Server and SQL Server Management Studio installed, and the sample database is already restored as the "SqlAssessment" database.

You should be able to open up SSMS and get started right away!

Frequently Asked Questions

  • I screwed up the database; how do I reset to the starting state?

    There is a .bak file in D:\TechAssessment\SQL\ that you can restore.
