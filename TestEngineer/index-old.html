Test Engineer tech assessment

How this works

As part of the assessment, you will write a few tests against a simple, web-based sample application. We will then perform a collaborative code review during a technical interview.

We know your time is valuable. We're only asking for 2 to 3 hours on this. If you can't finish the whole thing, that's OK! Do what you can.

You're welcome to spend more time on it if you want, but it's not required.

If you already have samples of your UI tests that you can share with us, those are acceptable too!

Starting the sample app

We have pre-installed the eShopOnWeb sample app on this virtual machine.

It should be automatically started and available at https://localhost:5001. There are bookmarks to this app in the Edge browser.

If the web app isn't running, check the c:\TechAssessment\__START_ESHOP.bat file for the sample instructions to get it running.

Instructions

We have created an empty C# + Nunit + Selenium project in c:\TestAccessment\TestEngineer\ for you to work in. You can open it with Visual Studio.

View the instructions on GitHub for the specific tests to write.

To run your tests, use the Test -> Run All Tests menu in Visual Studio.
