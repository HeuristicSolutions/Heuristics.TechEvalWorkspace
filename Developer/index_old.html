Full Stack Developer assessment

How this works

As part of the assessment, you will complete a few web development tasks in a sample project we have provided. will then perform a collaborative code review during a technical interview.

We know your time is valuable. We're only asking for 2 to 3 hours on this. If you can't finish the whole thing, that's OK! Do what you can.

You're welcome to spend more time on it if you want, but it's not required.

If you already have code samples that you can share with us, those are acceptable too!

Starting the sample app

There is a Visual Studio solution on this VM in c:\TechAssessment\Developer.

  1. Open it in Visual Studio
  2. Press Ctrl-F5 to run it
  3. Follow the on-screen instructions. You DO NOT need to do the "Initial Setup" task; it's been done already.
NOTE: This assessment does NOT use the "eShopOnWeb" sample app that is also installed on this VM! That one is part of the Test Engineer assessment; the sample app referred to here must be run through Visual Studio.
